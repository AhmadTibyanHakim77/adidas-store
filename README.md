# 👟 Adidas Store

> Modern E-Commerce Website built with **PHP Native**, **MySQL**, **HTML5**, **CSS3**, and **JavaScript**.

🌐 **Live Demo:** https://adidasstore.kesug.com

---

## 📖 Overview

Adidas Store is a modern e-commerce web application developed as a Full Stack Developer portfolio project. The application provides an online shopping experience through product browsing, user authentication, shopping cart, wishlist, and an admin dashboard for product management.

The project demonstrates responsive web development, clean code organization, and database integration using PHP Native and MySQL.

---

## ✨ Features

- Responsive Landing Page
- Product Catalog
- Product Detail
- Shopping Cart
- Wishlist
- User Login & Register
- User Profile
- Checkout System
- Contact Page
- Admin Dashboard
- Product Management

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AhmadTibyanHakim77/adidas-store.git
```

Go to project directory

```bash
cd adidas-store
```

Import the database

```text
database/adidas.sql
```

Configure the database connection

```text
includes/config.php
```

Run the project using **XAMPP**, **Laragon**, or another PHP server.
