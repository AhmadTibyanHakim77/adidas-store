# Adidas Premium Store

A modern PHP + MySQL e-commerce starter for XAMPP/Apache local development.

## Features
- Premium storefront landing page
- Product catalog and detail pages
- Cart, wishlist, checkout, login and registration
- Admin area for managing products/categories/orders
- SEO files and responsive design

## Setup
1. Start Apache and MySQL in XAMPP.
2. Import [database/adidas.sql](database/adidas.sql) into MySQL.
3. Place this folder inside htdocs.
4. Open http://localhost/adidas-store/.
